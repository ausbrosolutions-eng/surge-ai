# The Surge Ops Audit
## Productized Discovery Engagement for Restoration Contractors

**What it is:** A two-week paid diagnostic engagement that identifies stuck revenue, operational bottlenecks, and the automation roadmap to fix them. Built from the proven Rehab Restoration audit framework.

**Who it's for:** Restoration contractors doing $1M-$15M in annual revenue, with 5-50 employees, using JobNimbus, Albi, Xcelerate, PSA, iRestore, or JobDox as their primary CRM.

**Price:** $3,500 (one-time, paid upfront)

**Timeline:** 2 weeks from kickoff to final delivery

---

## The Offer

"Give us 90 minutes and access to your CRM. In 14 days, we will show you exactly where your business is losing money, what it would take to recover it, and a prioritized 90-day plan to build the infrastructure that does. If we can't identify at least $150,000 in recoverable annual revenue, we refund the full $3,500."

**For the client:**

- Implementation roadmap they can execute internally OR hand back to you
- No vendor lock-in, no multi-month commitment

## The Deliverable (What They Get)

A 15-25 page branded report PDF containing:

### Section 1: Executive Summary (2 pages)
- Total revenue at risk
- Cost of inaction per month
- Top 3 critical problems ranked by dollar impact
- Recommended implementation path

### Section 2: Live Data Audit (4-6 pages)
- Pipeline analysis by stage (jobs, revenue, average age)
- Collection crisis diagnostic (stuck invoices, days outstanding, worst offenders)
- Documentation gap analysis (jobs missing required attachments at billing stage)
- Data integrity score (lead source capture, job type classification, field completeness)
- Win/loss patterns (close rate, loss reasons, sales cycle length)

### Section 3: Current State Customer Journey Map (2-3 pages)
- Step-by-step map of their current lead-to-payment flow
- Identification of every manual handoff and system gap
- Where money leaks and where time leaks

### Section 4: Target State & Gap Analysis (3-4 pages)
- What "good" looks like for a restoration contractor their size
- Specific gaps between current state and target state
- Categorization by fix difficulty (quick wins vs. infrastructure vs. culture)

### Section 5: Tech Stack Recommendation (3-4 pages)
- Honest assessment of current CRM fit
- Recommended tool stack with pricing
- Comparison of 2-3 viable paths forward (keep current CRM + integrations, or migrate)
- ROI projections for each path

### Section 6: 90-Day Implementation Roadmap (2-3 pages)
- Days 1-30: Quick wins + foundation
- Days 31-60: Core automation build
- Days 61-90: Optimization + reporting
- Milestones, deliverables, and success metrics for each phase

### Section 7: Surge Implementation Proposal (1-2 pages)
- Phased retainer pricing if they want Surge to execute
- What's included in each phase
- Alternative: self-implementation playbook (at additional cost)

## The Process (2-Week Timeline)

### Week 1

**Day 1: Contract signed + invoice paid**
- Austin sends contract + Stripe invoice for $3,500
- Client completes 15-minute intake form (company size, revenue, current tools, pain points)
- Client grants API access to JobNimbus/Albi/etc. (API key or read-only user)

**Day 2-3: Kickoff Call (90 minutes, recorded)**
- Attendees: owner, ops manager, branch manager, lead admin (ideally 3-4 people)
- 20 min: Company overview, goals, seasonal patterns, competitive landscape
- 40 min: Full workflow walkthrough per role (lead intake, dispatch, documentation, billing, collections, reporting)
- 20 min: Pain point ranking, open questions
- 10 min: Next steps, what they should expect

**Day 4-5: Live data audit**
- Pull every open job via API
- Analyze pipeline by stage, revenue, age
- Identify stuck invoices and worst-case exposure
- Score data integrity (lead source, job type, required fields)
- Map documentation gaps against billing stage requirements

### Week 2

**Day 6-8: Analysis and report writing**
- Build current state journey map
- Identify target state gaps
- Research tech stack options for their specific size/trade
- Calculate ROI scenarios
- Draft implementation roadmap

**Day 9: Internal review**
- Check math twice
- Pressure-test recommendations against actual tool capabilities
- Finalize proposed path

**Day 10: Report delivery + 60-minute review call**
- PDF delivered 24 hours before call
- 60-minute walk-through with decision-makers
- Live Q&A, pressure-testing, refinement
- If moving forward with Surge: $3,500 credited toward first month of retainer

## Pricing Logic

**Why $3,500:**
- High enough to signal quality and filter tire-kickers
- Low enough to be a no-brainer for a contractor doing $2M+ in revenue
- Industry consulting rates: $250-$500/hr; 10-15 hrs of work = $2,500-$7,500 range
- Refund guarantee requires us to find $150K in recoverable revenue (43x return on audit fee, easy to exceed based on Rehab baseline)
- If we credit the $3,500 to retainer, it becomes a sunk-cost incentive to proceed

**Variations:**
- **Standard:** $3,500 for single-location restoration contractor under $5M
- **Multi-location:** $5,500 for 2-3 branches (more data, more stakeholders)
- **Enterprise:** $7,500 for 4+ branches or $10M+ revenue (custom scoping)
